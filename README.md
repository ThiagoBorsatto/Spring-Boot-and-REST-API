# Spring-Boot-and-REST-API
This is a Repo about Spring Boot in Java and REST API in Rust. The main goal is create a similar projects using the two frameworks and learner about yours differences and how they work.

First setup the Java and JDK to the last version available. Then setup the 'cargo' compiler for rust.

## Spring Boot in Java
Use the plugin of Spring Boot in VSCode to create the base for the API
    ctrl + Shift + P
To setup the Spring Boot basic body

## REST API in Rust
In rust just use the command on the terminal
    cargo new api_name
The basic body has been created
